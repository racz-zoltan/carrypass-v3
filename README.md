# CarryPass Legacy Archive

This repository contains legacy CarryPass files preserved for historical reference.

These files represent earlier versions, experiments, prototypes, and previous implementation stages of CarryPass. They are kept here to preserve project history, maintain backward reference material, and separate older code from the current active CarryPass repository.

## Status

**Archived / Legacy**

This repository is not the current CarryPass implementation.

The active CarryPass project has moved to a newer architecture and codebase, including:

- updated cryptographic key separation,
- current deterministic password generation logic,
- trusted-device based team vault access,
- revised documentation,
- updated security model,
- current PWA implementation.

The files in this repository may contain outdated code, obsolete assumptions, old UI patterns, previous cryptographic designs, deprecated workflows, or documentation that no longer reflects the current CarryPass model.

## Important Security Notice

Do not treat this repository as the authoritative or recommended version of CarryPass.

Legacy versions may not include the latest security improvements, architectural corrections, documentation updates, or implementation hardening present in the current CarryPass project.

If you are evaluating CarryPass for actual use, security review, deployment, or contribution, use the current active repository instead.

## Purpose of This Repository

This repository exists for:

- historical preservation,
- comparison with the current architecture,
- migration reference,
- transparency about project evolution,
- retaining old experiments without mixing them into the active codebase.

It is not intended for production use.
